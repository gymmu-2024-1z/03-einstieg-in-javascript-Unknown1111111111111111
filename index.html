Sortierverfahren

Bubblesort ist ein einfacher Sortierverfahren, der die Elemente in einer Liste in aufsteigender Reihenfolge sortiert. Das Bubbelsort Sortier ferfahren, tauscht immer 2 Elemente nebeneinander aus und setzt die kleinere Zahl nach vorne. So prüft es die ganze Reihe durch. Beispiel: [4, 3, 2, 1] -> [3, 4, 2, 1] -> [3, 2, 4, 1] -> [2, 3, 4, 1] usw. Bis es zum Endergebniss kommt -> [1, 2, 3, 4] Es muss schlussendlich n kleiner n + 1 So werden alle Zahlen nach Reihenfolge/ Grösse sortiert. Das gleiche gilt, aber auch für Buchstaben, die je nach Alphabetgrösse geordnet werden.
