Sortierverfahren

BUBBLESORT-SORTIERFERFAHREN

Bubblesort ist ein einfaches Sortierverfahren, das die Elemente in einer Liste in aufsteigender Reihenfolge sortiert.

Das BubbelsortSortierverfahren, tauscht immer 2 Elemente nebeneinander aus und setzt die kleinere Zahl nach vorne. So prüft es die ganze Reihe durch.

Beispiel: [4, 3, 2, 1] -> [3, 4, 2, 1] -> [3, 2, 4, 1] -> [2, 3, 4, 1] usw. Bis es zum Endergebniss kommt -> [1, 2, 3, 4]

Es muss schlussendlich n kleiner n + 1 sein. So werden alle Zahlen nach Reihenfolge/ Grösse sortiert. Das gleiche gilt, aber auch für Buchstaben, die je nach Alphabetgrösse geordnet werden. Es wird oft in Lehrzwecken benutzt, oder in kleinen Prototypen/ Tests, in denen es weniger um die Leistung geht, sondern darum, schnell eine funktionierende Lösung zu haben.

Namen-herleitung: Das Bubblesort Sortierverfahren heisst so, weil die grösseren Elemente wie "Blasen" (Bubbles) durch wiederholte Vergleiche und Vertauschungen schrittweise nach oben " steigen". Quasi zu ihrem Platz verschoben werden.

SELECTIONSORT-SORTIERFERFAHREN

Das Selectionsort Verfahren im JavaScript sortiert ein Array schrittweise ( Ein Array ist eine geordnete Sammlung von Werten), indem es wiederholt das kleinste (oder größte) Element aus dem unsortierten Teil des Arrays auswählt und an die korrekte Position im sortierten Teil verschiebt.

UNTERSCHIEDE:

Der Unterschied zwischen dem Selectionsort Verfahren und dem Bubblesort Verfahren liegt darin, dass Selectionsort Verfahren in jedem Durchgang das kleinste (oder größte) Element auswählt und geradewegs an die richtige Position bringt, während Bubblesort Verfahren benachbarte Elemente vergleicht und sie bei Bedarf vertauscht, um die größeren Elemente nach oben aufzublasen/umzutauschen. Das Selectionsort Verfahren benötigt also weniger Tauschvorgänge als das bubblesort Verfahren.

Jedoch sind beide Sortierverfahren nicht so effizient für grössere Arrays. (Mengen an Zahlen/ Buchstaben)

RADIXSORT-SORTIERFERFAHREN

Radixsort Sortierverfahren ist ein nichtvergleichbares Sortierverfahren, das speziell auf Zahlen basiert. Es sortiert die Ziffern der Zahlen von der niedrigsten bis zur höchsten Stelle (oder umgekehrt, je nach Wunsch des Aufgabengäbers) Es ist eine effiziente Methode, und sollte besonders bei großen Zahlen verwendet werden. Zum Beispiel verwendet man das RADIXSORT-SORTIERFERFAHREN oft in Datenbanken, da dort eine schnelle und stabile Sortierung erforderlich ist.

UNTERSCHIEDE:

Unterschied zu Bubblesort Sortierverfahren und Selectionsort Verfahren, im Gegensatz zu Bubblesort Verfahren und Selectionsort Verfahren, die beide auf Vergleichen und Tauschen basieren, funktionier Radixsort Vergleichferfahren so, dass es keine direkte Vergleiche hat, sondern ordnet die Elemente basierend auf den Ziffern/ oder Zeichen jeder Zahl.
